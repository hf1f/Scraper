# 🕸️ Web Scraper Project

A simple and efficient Python-based web scraper designed to extract books data and save the structured data into a local JSON file.

## 🚀 Features
- Extracts real-time data efficiently.
- Cleans and strips formatting from the extracted text.
- Exports the final structured output into an `all_books_database.json` file.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Requests, BeautifulSoup4
